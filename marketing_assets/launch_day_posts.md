## 🤖 Reddit (Option B) - r/SideProject or r/InternetIsBeautiful

**When:** Post this 1-2 hours after the Twitter post.
**Subreddit:** `r/SideProject` is great for indie hackers.

**Title:**
I built a "Circuit Breaker" because I was scared my AI agent would drain my wallet 💸🛑

**Body:**

Hey everyone,

I've been building autonomous agents recently, and I hit a major roadblock: **How do I let them pay for their own API usage or server costs without giving them my credit card or main private key?**

I was terrified that a hallucinating LLM would burn through $500 in API calls or gas fees overnight.

So, I built **iAgentPay**. It's a Python SDK that acts as a financial layer for AI agents.

**The cool part?**
It has a localized **"Capital Guard"** (logic gate) that hard-stops any transaction if it exceeds a daily limit you set (e.g., $10/day). Even if the AI tries to send $1000, the SDK rejects it locally before putting it on-chain.

**Features:**
*   Works on Ethereum (Base, Polygon) and **Solana**.
*   Supports USDC and even Memecoins (because why not?).
*   Fully Open Source (MIT).

I just launched it on Product Hunt today and would love to hear what you think about the security approach!
